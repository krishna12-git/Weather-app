# 🌦️ Weather App

A simple and responsive weather application built using HTML, CSS, and JavaScript. It fetches real-time weather data using a weather API and displays current weather conditions for any city.

## 🚀 Features

- 🔍 Search weather by city
- 🌡️ Current temperature
- 💧 Humidity
- 🌬️ Wind speed
- ☁️ Weather condition with icon
- 📱 Responsive design
- ⚡ Fast and easy to use

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Weather API
